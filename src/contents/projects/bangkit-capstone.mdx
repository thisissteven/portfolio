## Short Description ✏️

This project is a work of 6 team members, including me, 2 machine learning members, and 3 cloud computing members of the [Bangkit 2023 Cohort](https://grow.google/intl/id_id/bangkit/?tab=machine-learning). This app allows users to scan foods and showing their edibility as the scan result.

## Project Goals 🎯

Build simple, working and pretty user interfaces, with loading indicators and subtle transitions.

## Features 🔧

- Authentication & Authorization
- Camera Access
- History of scanned foods
- Scanned food details

## Technologies Used 💻

The app is built using Jetpack Compose, Android Studio, and Firebase.
